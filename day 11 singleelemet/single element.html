LeetCode 136 - Single Number Explanation

We are given the list: [1, 2, 3, 4, 1, 2, 3]

The goal is to find the number that appears only once using the XOR (^) operation.

Step Current Number XOR Operation Updated num Explanation
1 1 0 ^ 1 = 1 1 First number, so num becomes 1.
2 2 1 ^ 2 = 3 3 XOR with 2, now num becomes 3.
3 3 3 ^ 3 = 0 0 XOR with 3 cancels itself out, so num resets to 0.
4 4 0 ^ 4 = 4 4 XOR with 4, now num becomes 4.
5 1 4 ^ 1 = 5 5 XOR with 1, num becomes 5.
6 2 5 ^ 2 = 7 7 XOR with 2, num becomes 7.
7 3 7 ^ 3 = 4 4 XOR with 3 cancels it out, num becomes 4.
